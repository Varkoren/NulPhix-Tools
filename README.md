**NulPhix-HackTools** — это агрессивный фреймворк для сетевой разведки, OSINT и стресс-тестирования. Создан для тех, кому нужен быстрый результат в одном терминале без лишней мишуры. 

> **Warning:** Весь функционал предоставлен исключительно для образовательных целей. Автор не несет ответственности за неправомерное использование софта.

---

## ⚡ Основные модули

| Модуль | Описание | Технологии |
| :--- | :--- | :--- |
| **WIFI-SCANNER** | Полный аудит сети + Brute-force админок роутеров. | `Scapy`, `Requests` |
| **DDOS-ATTACK** | Асинхронный HTTP/S шторм с поддержкой прокси и разных методов. | `Aiohttp`, `Multiprocessing` |
| **PORT-SCANNER** | Ультрабыстрый сканер портов с Banner Grabbing. | `ThreadPool`, `Socket` |
| **SSH-BRUTE** | Пробив SSH серверов через кастомные словари и прокси. | `Paramiko` |
| **SHERLOCK** | Поиск аккаунтов на 400+ ресурсах и пробив номеров телефона. | `OSINT Engine V3` |

💎 Почему именно NulPhix-HackTools?
В мире существуют тысячи скриптов для пентеста, но NulPhix — это про баланс между экстремальной мощностью и простотой использования.

🚀 Ультимативная скорость (Async Engine)
В то время как обычные скрипты сканируют сайты по одному, наши модули Sherlock и DDoS работают на асинхронных движках aiohttp.

Результат: Скорость работы в 5-10 раз выше, чем у стандартных инструментов на requests.

🧠 Умный OSINT V3
Тебе больше не нужно вручную обновлять базы данных или скачивать новые JSON-файлы.

Авто-обновление: При каждом запуске скрипт сам подтягивает актуальные базы (470+ сайтов) из официальных репозиториев.

Точность: Мы внедрили проверку не только по статус-кодам, но и по содержанию страниц (Error Message Detection).

📱 Нативная поддержка Termux
Мы не просто "запустили" код на Android. Мы оптимизировали интерфейс под мобильные терминалы.

Стабильность: Весь функционал ПК-версии доступен в твоем кармане без вылетов.

Удобство: Минимум лишних движений — установка всех зависимостей одной командой.

💾 Система умных отчетов
NulPhix не просто «выплевывает» текст в консоль, который теряется при закрытии.

Экспорт: После каждого сканирования (порты или никнеймы) программа предлагает моментально сохранить результат в файл: [target]_osint.txt.

🕶️ Эстетика хакера
Мы верим, что качественный инструмент должен выглядеть круто.

Визуал: Кастомный ASCII-арт, Matrix-дождь и цветовая индикация статусов (OK/ERR/INFO) создают правильную атмосферу для работы.

## 🛠️ Полный гайд по установке

### 🖥️ На Компьютер (Windows / Linux)

1. **Установите Python**
   Скачайте актуальную версию с [python.org](https://www.python.org/downloads/). При установке на Windows обязательно поставьте галочку **"Add Python to PATH"**.

2. **Клонируйте репозиторий**
   Откройте терминал (CMD или PowerShell) и введите:
   ```bash
   git clone https://github.com/Varkoren/NulPhix-Tools.git
   ```
3. **перейдите в папку с софтом**
   ```bash
   cd NulPhix-Tools
   ```
Установите зависимости

```Bash
pip install -r requirements.txt
```
Запуск

```Bash
python main.py
```

Примечание: Для работы WiFi-сканера на Linux запускайте через 
```bush
sudo python main.py.
```

📱 Установка и запуск на Android (Termux)
Преврати свой смартфон в портативный инструмент для пентеста за 5 шагов.

1️⃣ Подготовка системы
Открой Termux и вставь команду ниже. Она обновит все пакеты и установит Python с Git:

```Bash
pkg update && pkg upgrade -y && pkg install python git libjpeg-turbo -y
```

2️⃣ Клонирование репозитория
```Bash
git clone https://github.com/Varkoren/NulPhix-Tools.git
```

3️⃣ Переход в директорию
Заходим в папку с проектом:

```Bash
cd NulPhix-Tools
```
4️⃣ Установка зависимостей
Устанавливаем все нужные библиотеки (Requests, Scapy, Aiohttp и др.) одной командой:

```Bash
pip install -r requirements.txt
```
5️⃣ ЗАПУСК ПРОГРАММЫ 

```Bash
python main.py
```
